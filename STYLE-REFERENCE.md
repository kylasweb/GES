# Quick Style Reference Guide

## Header Styles

### 1. Default Header
**Best For**: General e-commerce sites, professional business sites
**Features**:
- Full navigation menu
- Search functionality
- Shopping cart with badge
- Wishlist with badge
- User account menu
- Theme toggle
- Mobile hamburger menu
- Sticky positioning with shadow on scroll

**Design**: Classic horizontal layout with all features

---

### 2. Minimal Header
**Best For**: Clean, modern sites, minimalist brands
**Features**:
- Compact logo (GES)
- Essential navigation only
- Cart icon
- User icon
- Minimal spacing (h-14)

**Design**: Single-row, ultra-compact, no search or theme toggle

---

### 3. Centered Header
**Best For**: Fashion sites, lifestyle brands, portfolio sites
**Features**:
- Two-tier layout
- Top bar: Theme, Wishlist, Cart, User
- Main bar: Centered logo and navigation
- Balanced, symmetrical design

**Design**: Logo and nav centered, actions on top bar

---

### 4. Transparent Header
**Best For**: Landing pages, hero-heavy sites, creative agencies
**Features**:
- Starts transparent over hero
- Transitions to solid white on scroll
- Text color changes (white → dark)
- Fixed positioning
- Maintains all functionality

**Design**: Overlay effect with smooth transitions

---

### 5. Mega Menu Header
**Best For**: Large catalogs, multiple product categories
**Features**:
- Full-width dropdown menus
- Category organization
- Product icons in menu
- Featured items
- Quick links
- View all option

**Design**: Dropdown shows 3-column grid with icons

---

## Footer Styles

### 1. Default Footer
**Best For**: Comprehensive information sites, enterprise
**Features**:
- 4-column grid layout
- Company info with logo
- Social media links
- Quick links section
- Customer service links
- Contact information (Email, Phone, Address)
- Copyright notice

**Design**: Dark background (gray-900), structured columns

---

### 2. Minimal Footer
**Best For**: Simple sites, apps, minimalist designs
**Features**:
- Single-row layout
- Logo + company name
- Essential links only
- Copyright text
- Responsive centering

**Design**: Clean white background, compact height

---

### 3. Newsletter Footer
**Best For**: Content sites, blogs, marketing-focused sites
**Features**:
- Prominent newsletter signup (gradient bg)
- Email input with subscribe button
- Secondary links section
- Social media icons
- Contact info
- Call-to-action focused

**Design**: Green gradient banner, eye-catching newsletter section

---

### 4. Social Footer
**Best For**: Social media driven brands, influencer sites
**Features**:
- Large social media icons (64px)
- 5 platforms: Facebook, Twitter, Instagram, LinkedIn, YouTube
- Platform names under icons
- Hover animations (scale + color)
- Newsletter CTA card
- Minimal text links

**Design**: Dark background with colorful social buttons

---

## Quick Selection Guide

**Choose Default Header + Default Footer if**:
- You need all features
- Running an e-commerce store
- Want professional appearance
- Have many navigation items

**Choose Minimal Header + Minimal Footer if**:
- You want clean aesthetics
- Have limited navigation
- Prefer minimalist design
- Target mobile-first audience

**Choose Centered Header + Newsletter Footer if**:
- You're a fashion/lifestyle brand
- Content marketing is important
- Want balanced, elegant design
- Focus on email collection

**Choose Transparent Header + Social Footer if**:
- You have hero images/videos
- Building a landing page
- Social media is your main channel
- Want modern, dynamic look

**Choose Mega Header + Default Footer if**:
- You have many product categories
- Running a large catalog
- Need organized navigation
- B2B or wholesale business

## Implementation

All styles are available at `/admin/appearance`. Simply click "Select" on your preferred combination, and changes apply instantly across the entire site.

No code changes required!
